Lion UI Locked Selection Component

UPDATE IN PROGRESS!

Overview

The fl-lion-locked-selection component is a custom web component that extends Lion's LionListbox to implement locked selection functionality, similar to a quiz or test interface.

Key features:

  • Once an option is selected, it becomes locked, preventing further changes
  • Validates answers against a correct answer
  • Provides resolved state for each selection
  • Supports both vertical and horizontal layouts
  • Visually indicates correct answers

Further enhancements:

  • Disable selection after maximum failed selection

Implementation Details

This component extends the Lion UI listbox component to create a selection that becomes locked after the user makes a choice. It validates the selection against a correct answer and provides visual feedback. The component supports both vertical and horizontal layouts for different UI requirements.
