Lion-UI Locked select component

Overview

The fl-lion-locked-selection component is a custom web component that extends Lion's LionListbox to implement locked selection functionality, similar to a quiz or test interface.

Key features:

  • Once an option is selected, it becomes locked, preventing further changes
  • Validates selections against the answer
  • Provides resolved state for each selection
  • Supports vertical and horizontal layouts
  • Updates the option attributes for correct answer

Further enhancements:

  • Disable selection after maximum failed selection
  • Add checkbox style options

Implementation Details

This component extends the Lion UI listbox component to create a selection that becomes locked after the user makes a choice.

Component api

Styling

The lion-options are sitting in the light dom, hence styling can be done by simply targeting them from your app
