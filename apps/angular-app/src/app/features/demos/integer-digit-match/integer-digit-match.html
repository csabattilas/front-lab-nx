Lion-UI Integer digit match component

Overview

The fl-lion-integer-digit-match component is a custom web component that extends Lion's LionField to create a multi-input field for matching individual digits of a target digit list (not only integer number).

Key features:

  • Creates individual input fields for each digit
  • Supports left to right and right to left input order
  • Validates input against a target number
  • Restricts input to numeric values only
  • Provides visual feedback on validation
  • Supports dynamic target numbers

Further enhancements

Add more css variables to customize the input fields.

Implementation Details

This component uses the MultiInputMixin to create multiple input fields from a single component.

The integer-digit-match is implementing and extended class with input fields, each accepting a single digit and automatically focusing on the next field when a digit is entered.

The component validates the entire input against the target number and provides feedback.

Component api

Styling

Styling this component is done via a set of css variables and via ::part(input-group) pseudo-class

The list of css variables are as follows:

  • --fl-lion-integer-digit-match-digit-input-width
  • --fl-lion-integer-digit-match-input-invalid-color
  • --fl-lion-integer-digit-match-input-border-width
  • --fl-lion-integer-digit-match-input-color
