Checkbox tree form-control

Overview

The fl-form-checkbox-tree component is a regular checkbox tree, built using signals (Angular v20).

Future enhancements

The tree is composed of two tags a parent fl-form-checkbox-tree and a child fl-form-checkbox-tree-node

Implementation Details

The tree is composed of two tags a parent fl-form-checkbox-tree and a child fl-form-checkbox-tree-node

The parent tag will implement a custom formcontrol having values of number[]

Updating the states of the parent nodes is done via "registering" to the viechildren signal changes

Updating the control value is done internally via a context signal managed by the control fl-form-checkbox-tree component

Component api (tree node)

Styling

As usual, with angular component/global styling
