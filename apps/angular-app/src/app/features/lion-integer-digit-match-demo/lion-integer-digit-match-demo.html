Lion UI Integer Digit Match Component

WORK IN PROGRESS!

The fl-lion-integer-digit-match component is a custom web component that extends Lion's LionInput to create a multi-input field for matching individual digits of a target number.

Key features:

  • Creates individual input fields for each digit
  • Validates input against a target number
  • Restricts input to numeric values only
  • Provides visual feedback on validation
  • Supports dynamic target numbers

Example 1: Basic Usage

This example shows the basic usage with a 4-digit target number:

  • The target property is set to "4829"
  • Four individual input fields are automatically created
  • Each input field accepts only a single digit
  • Input is validated against the target number

Enter the number: 4829

Component API

Properties

Property Type Description
target String The target number to match (e.g., '4829')
modelValue String The current input value as a concatenated string
segments Array Internal array of target digits (e.g., ['4','8','2','9'])

Events

Event Description
model-value-changed Fired when any digit input changes

Implementation Details

This component uses the MultiInputMixin to create multiple input fields from a single component. Each input field is restricted to a single digit and automatically focuses on the next field when a digit is entered. The component validates the entire input against the target number and provides feedback.
